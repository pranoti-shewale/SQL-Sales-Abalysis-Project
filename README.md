SQL Sales Analysis Project

Project Overview

This project analyzes retail sales data using SQL queries.
The goal is to identify sales trends, profitable products, customer insights, and regional performance.

Tools Used

- MySQL
- SQL
- Excel Dataset

Dataset

The dataset contains sales information such as:

- Order ID
- Order Date
- Customer Name
- Product Name
- Sales
- Quantity
- Profit
- Region

Dataset File: "superstore_sales.csv"

Key SQL Analysis

The following analysis was performed using SQL queries:

1. Total Sales Analysis
2. Region Wise Sales Performance
3. Top Selling Products
4. Profit Analysis by Category
5. High Revenue Products
6. Loss Making Products
7. Top Customers by Sales
8. Monthly Sales Trend

Key Insights

- Identified top revenue generating products.
- Analyzed region wise sales performance.
- Found customers with highest purchase value.
- Discovered products generating losses.
