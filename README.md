Sales Data Analysis using SQL

Project Overview

This project analyzes sales data using SQL queries to identify business insights such as sales trends, top performing products, customer purchasing behavior, and regional sales performance.

Tools Used

- MySQL
- SQL
- CSV Dataset

Skills Demonstrated

- Data Analysis using SQL
- Aggregate Functions
- GROUP BY
- HAVING
- Subqueries
- Data Exploration

Dataset

The dataset contains sales information such as:

- Order ID
- Order Date
- Customer Name
- Product Name
- Sales
- Quantity
- Profit
- Region

Dataset File: superstore_sales.csv

Key SQL Analysis

1. Total Sales Analysis
2. Region Wise Sales Performance
3. Top Selling Products
4. Profit Analysis by Category
5. High Revenue Products (Using HAVING Clause)
6. Loss Making Products
7. Top Customers by Sales
8. Monthly Sales Trend

Key Insights

- Identified top revenue generating products.
- Analyzed regional sales performance.
- Identified high value customers based on total sales.
- Detected loss making products affecting overall profitability.
